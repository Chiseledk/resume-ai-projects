# 语音门禁系统 - Web版

基于 Flask + DashScope 的网页端语音门禁系统，支持声纹注册与双重验证（声纹+口令），纯网页模拟，无需硬件。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple("清华镜像源")
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问

浏览器打开：**http://127.0.0.1:5000**

---

## 📖 使用流程

### 首次使用：注册声纹

1. 点击 **A. 进入注册**
2. 输入管理员密码：运行前通过 `ADMIN_PASSWORD` 环境变量配置
3. 点击麦克风按钮，对麦克风清晰说出"**开门**"
4. 等待 3 秒，系统自动保存声纹
5. 显示注册成功，获得用户 ID

### 日常使用：验证开门

1. 点击 **B. 进入监听**
2. 点击麦克风按钮，对麦克风说出"**开门**"
3. 系统进行双重验证：
   - ✅ **声纹匹配 + 口令正确** → 门动画打开 + 成功音效
   - ❌ **声纹不匹配** → 红灯闪烁 + 失败音效
   - ❌ **口令错误** → 红灯闪烁 + 失败音效

## 🔑 核心函数

| 功能 | 函数 | 说明                       |
|------|------|--------------------------|
| 接收音频 | `request.files["audio"]` | Flask 接收前端上传             |
| 提取声纹 | `extract_voiceprint()` | librosa 提取 13 维 MFCC     |
| 保存声纹 | `save_voiceprint()` | 保存到 data/models/{id}.npy |
| 声纹匹配 | `find_match_user()` | 余弦相似度比对                  |
| 语音识别 | `recognize_command()` | DashScope 识别"开门"/本地识别"开门" |

---

## 🗣️ 本地识别原理/方法
| 步骤    | 使用技术             | 原理说明                         | 对应代码核心变量                            |
|-------|------------------|------------------------------|-------------------------------------|
| 音频预处理 | 波形规整、声道统一        | 读取语音波形，统一采样率，立体声转为单声道，消除录制干扰 | y波形数据、sr采样率                         |
| 多特征提取 | 过零率、频谱质心、短时能量、时长 | 抓取人声四类物理属性，区分人声、噪音、不同语句      | `zcr、centroid、rms、duration`           |
| 口令判定  | 阈值规则匹配           | 对照标准开门口令特征区间，筛选合法语音          | `is_open_command`判定标识                 |
| 声纹提取  | MFCC梅尔倒谱算法       | 模拟人耳听觉特性，生成个人专属声音指纹向量        | `current_feat`实时声纹特征                  |
| 身份比对  | 向量点积相似度算法        | 匹配注册声纹库，计算音色相似程度             | `max_sim`最高相似度值                       |
| 结果输出  | 双重条件决策           | 口令与声纹均满足标准则授权开门，反之判定识别失败     | `VOICEPRINT_SIMILARITY_THRESHOLD`判定阈值 |

## ⚙️ 配置参数

编辑 `config.py`：

| 参数                                | 说明                  | 默认值        |
|-----------------------------------|---------------------|------------|
| `ADMIN_PASSWORD`                  | 管理员密码               | 通过环境变量配置 |
| `DASHSCOPE_API_KEY`               | 阿里云 DashScope API Key | 通过环境变量配置 |
| `VOICEPRINT_SIMILARITY_THRESHOLD` | 声纹相似度阈值 (0-1)       | `0.85`     |
| `RECORD_SECONDS`                  | 注册录音时长              | `3` 秒      |
| `SAMPLE_RATE`                     | 音频采样率               | `16000` Hz |
| `ASR_ENGINE `                     | 语音识别方法              | `3`种（可切换）  |

## ❓ 常见问题

### Q: 提示"Format not recognised"错误？

前端已直接生成 WAV 格式，无需转换。确保使用最新版本代码。

### Q: 声纹验证总是失败？

1. 检查注册时和验证时是否使用同一设备/浏览器
2. 说话时保持环境安静
3. 距离麦克风 10-20cm
4. 可以调低 `VOICEPRINT_SIMILARITY_THRESHOLD` 到 `0.75`

### Q: 如何删除已注册用户？

删除 `data/models/` 目录下对应的 `.npy` 文件即可。

### Q: ASR 识别失败？

1. 检查 `DASHSCOPE_API_KEY` 是否正确配置
2. 检查网络连接
3. 查看后端控制台日志
